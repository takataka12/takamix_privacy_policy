プライバシーポリシー（日本語）

TAKA MIX（以下「本アプリ」）は、歌ってみたMIX依頼の受付を目的としたiOSアプリです。 本プライバシーポリシーは、本アプリにおける利用者情報の取扱いについて定めるものです。

1. 取得する情報

  • 楽曲名、音源URL、希望納期、Key/BPM、MIXの要望
  • X（旧Twitter）ID（任意）
  • 受付番号（アプリが自動生成）

本アプリは、広告識別子（IDFA）等のトラッキング情報を取得しません。

2. 利用目的

  • MIX依頼の受付・管理・進行
  • 依頼者への連絡
  • 不正防止および問い合わせ対応

3. 外部送信・第三者提供

  • Discord（Webhook）
    依頼内容は、開発者が管理するDiscordサーバーに送信されます。

決済はAppleのアプリ内課金（StoreKit）を利用します。 決済情報はAppleが管理し、開発者がカード情報等を取得することはありません。

4. 保管期間

依頼対応に必要な期間、情報を合理的な範囲で保存し、不要となった場合は削除します。

5. お問い合わせ

メール：suparnaturalz@gmail.com

Privacy Policy (English)

TAKA MIX (the “App”) is an iOS application designed to accept song mixing requests. This Privacy Policy explains how user information is handled.

1. Information Collected

  • Song title, audio source URL, requested delivery date, key/BPM, notes
  • X (formerly Twitter) ID (optional)
  • Automatically generated receipt number

2. Purpose of Use

  • Processing and managing mix requests
  • Contacting users regarding their requests
  • Preventing abuse and responding to inquiries

3. External Services

  • Discord (Webhook)
    Request details are sent to a Discord server managed by the developer.

Payments are processed by Apple via In-App Purchase. The developer does not receive payment credentials.
